# [Backstage](https://backstage.io)

This is your newly scaffolded Backstage App, Good Luck!

To start the app, run:

```sh
yarn install
yarn dev
```

## To run this on your local Kubernetes cluster

run - 

```console
# from root of the repo
$ tilt up
```

and navigate to localhost:7007
